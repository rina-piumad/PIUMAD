# 🚀 Pium Homepage 개발 가이드

## 📦 서버 실행 방법

### 방법 1: 배치 파일 사용 (가장 쉬움)

1. `dev.bat` 파일을 **더블클릭**
2. 끝! 브라우저에서 http://localhost:3000 접속

### 방법 2: 터미널 사용

```bash
# 프로젝트 폴더에서
npm run dev
```

### 주요 명령어

| 명령어 | 설명 |
|--------|------|
| `npm run dev` | 개발 서버 실행 (http://localhost:3000) |
| `npm run build` | 프로덕션 빌드 |
| `npm run start` | 프로덕션 서버 실행 |
| `npm run lint` | 코드 검사 |
